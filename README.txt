To set up the database:

1. Start MYSQL server on XAMPP

2. Go to localhost/phpmyadmin

3. Click on 'New' icon to create a new database

4. Create a new database called "bartending"

5. Click the new bartending database

6. Click the 'Import' icon

7. Click 'Choose File' and open the bartending.sql File

8. Click the 'Go' button on the bottom right side of the screen to import database tables

9. Database functions should work on the site
